Różne skróty klawiszowe.

  • CTRL + X: wycinianie zawartości
  • CTRL + C: skopiowanie zawartości do schowka.
  • CTRL + V: wklejenie zawartości z schowka.
  • ALT + F4: zamknięcie danego programu.
  • WIN + SHIFT + S: screen tylko pewnego kawałka ekranu.
  • ALT + TAB: Szybkie przejście do ostatniego otwartego okna.
  • CTRL + F: szukanie danego wyrazu.
  • CTRL + Z: cofnięcie ostatniej akcji.
  • CTRL + N: otwarcie nowego okna eksploratora windows.
